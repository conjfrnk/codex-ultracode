# Ultracode-Style Capability Matrix

This project targets Ultracode-style orchestration, but it should not claim measured superiority without benchmark evidence on identical tasks, models, repositories, and budgets.

See [Claude Code / Ultracode Capability Inventory](claude-code-capabilities.md) for the sourced target matrix checked on 2026-07-13.

Live Claude/Ultracode runs may consume paid model budget. Claude Code 2.1.203+ documents `--effort ultracode` for noninteractive sessions; every comparison must still pin the model, cap budget/turns, and confirm the resolved model in captured output.
