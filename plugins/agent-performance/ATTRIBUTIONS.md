# Provenance

These skills, packet schemas, validator, and outcome reducer are original local work. They
synthesize broadly used engineering practices: explicit behavior ownership, decision continuity,
causal debugging, behavioral testing, evidence-backed review, invariant-preserving simplification,
completion proof, and blinded matched comparisons.

No third-party runtime extension, copied template, or host-specific code is included. The plugin is
distributed under the repository's MIT license.
